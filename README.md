# php-go-round

PhpGoRound

Start container:

```
docker-compose up --build -d
```

By default app should be available @ `localhost:50000`

Or if already built:

```
docker-compose up -d
```

Stop container:

```
docker-compose down
```

Rebuild container:

```
docker-compose build
```

## Extras

Enter the shell:

```
docker/shell
```
